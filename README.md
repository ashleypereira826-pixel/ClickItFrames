# Photography Portfolio

A basic photography portfolio website built with Vite, HTML, CSS, and JavaScript.

## Features

- Responsive navigation with dropdown menu
- Separate dedicated pages for Portrait and Product photography
- Independent sliding carousels with animations
- Curved edges and enhanced shadows on images
- Smooth slide animations from right to left
- About and Contact sections

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```

4. Preview the production build:
   ```
   npm run preview
   ```

## Customization

Each gallery page has its own carousel. Replace the placeholder image URLs with your actual photos. Update the content in the About and Contact sections as needed. Each carousel runs independently with smooth fade animations and auto-advances every 2 seconds. Navigation uses React Router for seamless page transitions.
